Proposito del sitio

Este sitio tiene como proposito fortalecer el aprendizaje acerca de recursos basados en HTML5, CSS3 entre otras tecnologias orientadas a los conceptos de etiquetas basicas
de HTML, atributos y valores que son de gran importancia para el diseño de sitios web

Introduccion

Esta pagina web cuenta con 6 paginas las cuales se describen a continuacion:

  • La primera pagina que es en la nos encontramos y la cual se denomina "pagina de inicio" muestra un breve resumen de los temas que se exponen y esta el menu para navegar entre los diferentes contenidos HTML que existen en este sitio web

  • La segunda pagina denominada "Tema 1: etiquetas basicas de HTML" muestra la informacion referente a las etiquetas basicas de HTML con sus respectivas imagenes de apoyo

  • La tercera pagina denominada "Tema 2: atributos y valores HTML" muestra la informacion referente a los atributos y valores de HTML con sus respectivas imagenes de apoyo

  • La cuarta pagina denominada "Recurso 1: etiquetas basicas de HTML" muestra la informacion referente a las etiquetas basicas de HTML mediante recursos multimedios

  • La quinta pagina denominada "Recurso 2: atributos y valores HTML" muestra la informacion referente a los atributos y valores de HTML mediante recursos multimedios

  • En la sexta y ultima denominada "pagina de autor" muestra la informacion del autor de esta pagina web

    • Hay que decir que en las paginas de aprendizaje no existe ningun tipo de contenido ya que por el momento no estan disponibles
